这里发生了什么？

  1. 你已登录 OA（账号存在 oa_users 表，密码 BCrypt 加密）。
  2. 点击「IAM 系统」→ 浏览器访问 /sso/launch（Spring Boot Controller）。
  3. Controller 检测 OIDC session：
      • 已有 → 直接跳 /dashboard.html（零感知）
      • 没有 → 跳 Keycloak，用 你的邮箱 登录
  4. Keycloak 把 email 写入 id_token，Spring Boot 建立 OIDC session。
  5. IAM dashboard 用 token 里的 email 在 users 表查找匹配的 IAM 档案。

💡 email 是跨系统的唯一标识：OA 用户表 → Keycloak 账号 → IAM 用户表，三者靠同一个 email 串联。
